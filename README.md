# HelloWorldMAD26 - Week 2

## Description
Welcome to the first mobile app of MAD: HelloWorld
This app shows a request permission to access and grant location permissions. If accepted, it shows info of GPS location.

## Key Features (Week 2)
* [cite_start]**Advanced Navigation**: Implementation of a 3-level navigation flow (MainActivity ↔ SecondActivity ↔ ThirdActivity)
* [cite_start]**GPS Location**: The app requests runtime permissions and displays the device's latitude and longitude
* [cite_start]**System Logging**: Integrated Logcat messaging with different levels of verbosity (Debug and Info) to monitor the activity lifecycle
* [cite_start]**Custom Styling**: UI components have been customized
